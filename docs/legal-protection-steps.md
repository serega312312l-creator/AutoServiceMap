1. Пошук схожих ТМ: [Укрпатент / IP офіс](https://ukrpatent.org/) (або актуальний портал УКРНОІВІ)
2. Перевірити Play Store / App Store на клони назви
3. Обрати класи (орієнтовно: ПЗ, мобільні додатки, онлайн-сервіси — уточнить повірений)
4. Подати заявку на словесну ТМ «AVTOGID» (+ окремо логотип, якщо є)
5. Після реєстрації — використовувати знак ® лише після отримання свідоцтва (до того можна ™)
6. Зберігати свідоцтво PDF у безпечному місці (не в публічному git)
