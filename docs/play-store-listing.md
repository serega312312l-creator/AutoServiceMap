# AVTOGID — тексти для Google Play

Скопіюй у Play Console → **Main store listing** → мова **Українська (uk)**.

---

## Назва додатку (до 30 символів)

```
AVTOGID
```

## Короткий опис (до 80 символів)

```
СТО та допомога на дорозі. Працює офлайн — сервіс, евакуатор, шини поруч.
```

## Повний опис (до 4000 символів)

```
AVTOGID — помічник водія для дороги. Знайдіть СТО, евакуатор і шиномонтаж навіть без інтернету. Коли сталася поломка — екран «Поломка» і маршрут до найближчого сервісу.

ГОЛОВНЕ

• Офлайн-база СТО у вашому регіоні
• Карта та список сервісів поруч (фільтри: СТО, шини, АЗС, евакуатор…)
• Екран «Поломка» — 112, евакуатор, найближче СТО
• Маршрут у додатку, попередження «мертвих зон» без сервісів
• Зупинки в дорозі: відпочинок, їжа й готелі вздовж маршруту

PREMIUM (пробний період)

• Офлайн-карти цілого регіону
• Гараж, нагадування ТО, SOS сім’ї
• Альтернативні маршрути та голосові підказки

Платна підписка через Google Play з’явиться в наступному оновленні. Зараз можна спробувати Premium безкоштовно на 7 днів.

ДЛЯ КОГО

Для водіїв у міжміських поїздках і в незнайомому районі — коли потрібен сервіс швидко й надійно.

КОНФІДЕНЦІЙНІСТЬ

Геолокація використовується для показу місць поруч. Акаунт і фото — лише за вашим бажанням. Деталі — у політиці конфіденційності.

Підтримка: serega312312l@gmail.com
```

## Перед продажем підписки в наступному релізі

1. Увімкнути `PREMIUM_BILLING_ENABLED = true` у `src/constants/premium.ts`
2. Створити підписки в Play Console:
   - `avtogid_premium_monthly`
   - `avtogid_premium_yearly`
3. Підключити `react-native-iap` / Billing у production-збірці
4. Internal testing → перевірити покупку → Production
